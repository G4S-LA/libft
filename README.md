# libft
Целью данного проекта является собственная библиотека, включающая в себя следующие функции:
### :mag: Проверка/редактирование символа
- `ft_isalnum`
- `ft_isalpha`
- `ft_isascii`
- `ft_isdigit`
- `ft_islower`
- `ft_isprime`
- `ft_isprint`
- `ft_isspace`
- `ft_isupper`
- `ft_toupper`
- `ft_tolower`
